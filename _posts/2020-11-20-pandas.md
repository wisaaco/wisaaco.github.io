# Pivot a table

Code:
```python
dem = dem.groupby(["time","action"]).agg({"action":"count"})
dem.columns=["freq"]
dem.reset_index(level=["time","action"], inplace=True)
dem = dem.pivot_table('freq', ['time'], 'action')
```

Input
```csv
                  freq
time   action         
300    replicate     6
500    migrate       1
       nop           1
       replicate    10
700    migrate       1
               ...
100500 nop          34
100700 none          2
       nop          35
100900 none          2
       nop          35
```
